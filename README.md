# 📌 Sistema CRUD em Java

Este é um projeto pessoal desenvolvido com o objetivo de praticar conceitos fundamentais de desenvolvimento backend utilizando Java, com foco na implementação de um sistema CRUD (Create, Read, Update e Delete).

O projeto simula operações básicas de manipulação de dados, que são a base de praticamente qualquer sistema moderno.

---

## 🚀 Funcionalidades

- Criar registros
- Listar dados cadastrados
- Atualizar informações existentes
- Deletar registros

---

## 🛠️ Tecnologias utilizadas

- Java
- Programação Orientada a Objetos (POO)
- Manipulação de dados em memória/arquivo
- Estrutura em camadas simples

---

## 📂 Estrutura do projeto

- `Main` → Inicialização da aplicação  
- `Service` → Regras de negócio  
- `Model` → Representação dos dados  
- `Menu/Usuário` → Interação com o usuário
